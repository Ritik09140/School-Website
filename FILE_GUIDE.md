## 🆘 Emergency Help

### Website Won't Load
1. Check if `index.html` exists
2. Open in browser (no Live Server needed)
3. Try different browser
4. Clear cache (Ctrl+Shift+Delete)

### Images Not Showing
1. Check file paths in HTML
2. Ensure image files exist
3. Check folder structure
4. Use relative paths: `../images/photo.jpg`

### Live Server Not Working
1. Restart VS Code
2. Reinstall Live Server extension
3. Use browser directly
4. Manual port: `http://127.0.0.1:5500`

### Git Not Working
1. Install Git: https://git-scm.com/download
2. Restart Terminal
3. Check username/email config
4. Try different terminal (PowerShell, Git Bash)
